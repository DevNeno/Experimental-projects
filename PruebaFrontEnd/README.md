## Descripción
Proyecto experimental desarrollado en **HTML y CSS** que busca recrear la ventana de login de **Facebook**.  

El objetivo principal fue practicar la maquetación, estilos y posicionamiento de elementos en una página web, así como reproducir interfaces existentes de manera visual.

## Funcionalidades
- Interfaz de login visualmente similar a Facebook
- Posicionamiento y estilos de formularios y botones

## Tecnologías
- HTML
- CSS

## Aprendizajes y logros
- Práctica en maquetación y diseño de interfaces web
- Manejo de estilos CSS para reproducir una ventana de login
- Comprensión de estructura de formularios y diseño visual responsive básico

## Estado del proyecto
⚠️ **Prototipo / prueba inicial**  

Este proyecto se mantiene como referencia de aprendizaje y experimentación con diseño frontend.
