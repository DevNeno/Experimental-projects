## Descripción
Proyecto desarrollado en **Bash** para crear respaldos de directorios en un único archivo comprimido `.tar.gz`.  

El script incluye:  
- Modo verbose para seguimiento de la operación  
- Generación de logs de respaldo  
- Integración con **help2man** para crear el archivo `man` y documentar la herramienta como un comando de línea de comandos  

Este proyecto fue concebido como práctica de scripting y manejo de archivos, con enfoque en automatización de tareas y documentación de comandos.

## Funcionalidades
- Respaldo de directorios en archivos `.tar.gz`
- Registro de logs de ejecución
- Modo verbose para seguimiento
- Documentación automática mediante `help2man`

## Tecnologías
- Bash / Shell Scripting
- Tar / gzip
- help2man

## Aprendizajes y logros
- Automatización de procesos de respaldo mediante scripts
- Uso de flags, logs y modos verbose en Bash
- Generación de documentación manual (`man`) para herramientas personalizadas
