# Portafolio de Proyectos Personales

![Estado](https://img.shields.io/badge/estado-experimental-lightgrey)

## Descripción
Este repositorio reúne proyectos personales y experimentales desarrollados en distintas áreas de programación.  

Aunque algunos proyectos son prototipos iniciales o están incompletos, cada uno refleja aprendizajes en tecnologías, técnicas de desarrollo y conceptos explorados durante su creación.

El objetivo de este repositorio es mostrar la evolución de habilidades y curiosidad técnica, más allá de la finalización completa de cada proyecto.

## Proyectos

| Proyecto                  | Tecnología         | Descripción breve |
|----------------------------|-----------------|-----------------|
| Motor Gráfico OpenGL       | C++, OpenGL      | Prototipo inicial de motor gráfico; exploración de renderizado. |
| Prueba Allegro             | C++, Allegro     | Ventana con dos cuadrados, uno controlable con flechas; práctica de gráficos 2D y entrada de usuario. |
| Chat P2P Python            | Python           | Prototipo de chat P2P; experimentación con hilos y sockets, comunicación parcial entre dos nodos. |
| Prueba Login Frontend      | HTML, CSS        | Recreación de la ventana de login de Facebook; práctica de maquetación y diseño visual. |
| Script de Respaldo Bash    | Bash             | Script para respaldos en `.tar.gz` con modo verbose, logs y archivo man generado con help2man. |
