# Motor Gráfico Básico en C++ y OpenGL

![Estado](https://img.shields.io/badge/estado-prototipo%20inicial-orange)

## Descripción
Proyecto experimental para desarrollar un motor gráfico desde cero en C++ utilizando OpenGL.  
El objetivo principal era abrir una ventana y permitir renderizado básico de triángulos. Gran parte del avance se perdió debido a un borrado accidental, por lo que el repositorio actual contiene solo archivos iniciales y fragmentos de código.

## Tecnologías
- C++
- OpenGL

## Funcionalidades
- Renderizado básico de triángulos
- Configuración inicial del motor gráfico

## Aprendizajes y logros
- Organización de un motor gráfico desde cero
- Experimentos iniciales con OpenGL y manejo de contexto gráfico

## Estado del proyecto
⚠️ **Abandonado / prototipo inicial**  

El proyecto no está activo actualmente y sirve como referencia experimental.
