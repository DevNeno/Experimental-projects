## Descripción

Proyecto experimental desarrollado en C++ utilizando la biblioteca **Allegro**.

Permite abrir una ventana, renderizar dos cuadrados en pantalla y controlar uno de ellos mediante las teclas de flecha.

Este proyecto fue concebido como una prueba de conceptos para el manejo de gráficos y entradas en tiempo real. Aunque no evolucionó más allá de esta funcionalidad básica.

## Funcionalidades

* Creación de ventana gráfica
* Renderizado de objetos 2D (cuadrados)
* Control de un objeto mediante teclado
* Actualización en tiempo real

## Tecnologías

* C++
* Allegro

## Aprendizajes y logros

* Manejo de ventanas y renderizado 2D con Allegro
* Implementación básica de control de objetos mediante teclado
* Estructuración inicial de un proyecto interactivo en C++

## Estado del proyecto

⚠️ **Abandonado / prototipo inicial**

Este repositorio se mantiene como referencia de aprendizaje y prueba de conceptos.

## Cómo ejecutar

1. Clonar el repositorio:

   ```bash
   git clone https://github.com/usuario/prueba-allegro.git
   ```
2. Instalar Allegro en tu sistema.
3. Compilar el proyecto:

   ```bash
   make run
   ```


## Notas

Este proyecto puede servir como punto de partida para desarrollos más complejos en gráficos 2D o videojuegos simples.
