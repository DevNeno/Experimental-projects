# Chat P2P Experimental

![Estado](https://img.shields.io/badge/estado-prototipo%20inicial-orange)
![Lenguaje](https://img.shields.io/badge/lenguaje-Python-blue)

## Descripción
Proyecto experimental desarrollado en **Python** con el objetivo de crear un **chat P2P**.  

Explora conceptos de concurrencia mediante hilos y manejo de sockets para controlar las conexiones entre nodos. La versión actual funciona de manera parcial y presenta errores relacionados con la concurrencia, pero permite probar la comunicación entre dos nodos ejecutados en la misma computadora.

Este proyecto representa una primera aproximación a sistemas de comunicación P2P y programación concurrente.

## Funcionalidades
- Comunicación básica entre nodos en la misma máquina
- Concurrencia mediante hilos (`threading`)
- Manejo de sockets TCP/IP
- Pruebas de conexión P2P

## Tecnologías
- Python
- Sockets
- Hilos / threading

## Aprendizajes y logros
- Primeras aproximaciones a concurrencia y sincronización de hilos
- Manejo básico de sockets para comunicación P2P
- Diseño de pruebas de conexión entre múltiples nodos en la misma máquina

## Estado del proyecto
⚠️ **Abandonado / prototipo inicial**  

Se mantiene como referencia de aprendizaje y pruebas experimentales de comunicación P2P.
